# VFX Catalog

**Owner:** vfx-artist
**Status:** Draft
**Last Updated:** —

> Index of all available VFX prefab scenes. Agents needing to use a VFX effect should reference this catalog for the scene path and export variables.

---

## Format

| Field | Description |
|-------|-------------|
| Name | Effect name (matches scene root node name) |
| Scene Path | `game/scenes/vfx/<category>/<EffectName>.tscn` |
| Category | combat / environment / character / ui |
| Description | What it looks like and when to use it |
| Export Variables | Exposed parameters and their types/ranges |
| Particle Count | Approx. peak particle count |
| Performance Tier | Low / Medium / High |
