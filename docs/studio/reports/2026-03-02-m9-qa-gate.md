## Scope Delivered

### Root Game
- Canonical `game.tscn` root scene set as Godot project main scene; routes to DebugLauncher
  (debug builds) or MainMenu (release)
- `main_menu.tscn` — minimal main menu with centered Play button
- `game_world.tscn` — replaces TestWorld; reads `Global.starting_biome` and
  `Global.starting_inventory`; instantiates correct biome; resets game state on load
- `Global.starting_biome` / `Global.starting_inventory` props added
- `DebugLauncher` reduced from ~519 to ~159 lines — writes startup params, hands off to
  MainMenu → GameWorld; Begin Wealthy now grants full stacks per resource max
- `TestWorld` removed; all tests confirmed to use direct instantiation
- `Global.log()` → `Global.debug_log()` renamed across 49 files / ~280 call sites (fixed
  conflict with Godot 4.5 built-in `log(float)` that broke headless mode)

### Orchestrator Resilience
- Dead-lock fix for IN_PROGRESS pre-claim; silent-success detection on crash handler
- Checkpoint system — writes/reads/clears suspension state; resume dispatch injects context
- Usage-limit detection + LIMIT_WAIT cooldown
- UID commit idempotency — `_handle_uid_commits()` restartable after interruption
- Structured suspension logging; gate deferral when unresolved checkpoints exist
- Conductor-level gate detection fallback when Producer unavailable
- Auto-remediation for silently-merged PRs with IN_PROGRESS tickets
- Log archive rotation at milestone close
- Resilience runbook (`docs/engineering/orchestrator-resilience-runbook.md`); CLAUDE.md
  Suspension & Resume section and updated "On Milestone Close" checklist

### Gamepad Bugs
- Left stick Y-axis corrected (push up = forward)
- Right stick sensitivity raised (`gamepad_sensitivity_x=15.0`, `gamepad_sensitivity_y=7.0`)
- `interact` action bound to `JOY_BUTTON_A` (Xbox A / PlayStation Cross)
- Interaction prompt HUD dynamically switches key badge and headlamp hint on device change
- `InventoryActionPopup` wired into inventory screen (Y button, grid pause, hint bar, gamepad
  controls descriptor)
- Gamepad popup action buttons now function correctly (TICKET-0270)
- Gamepad B button cancels the item actions popup (TICKET-0271)

### Code Quality
- Section header mislabeling fixed in 4 M8 data classes
- `DeepResourceNode.new()` adopted in all biome scripts (replaces `Deposit.new(infinite=true)`)
- `move_and_slide()` moved to `_physics_process()` in `PlayerFirstPerson`
- Null guard added to NavigationConsole test `after_each()`

### Gameplay Polish
- Drop items from inventory as physical world objects; pickup via interact; despawn on biome change
- Destroy items from inventory with confirm dialog (CANCEL focused by default)
- Debug Launcher Fast Move (3×) toggle; gated behind `OS.is_debug_build()`
