# M2 PoC Experiment Archive

**Archived:** 2026-02-22
**Milestone:** M2 — 3D Asset Pipeline
**Archived by:** technical-artist (TICKET-0018)

---

## Purpose

This directory preserves all Proof-of-Concept artifacts from M2 pipeline evaluation. These assets were generated during TICKET-0009 (Blender Python PoC) and TICKET-0010 (AI Generation PoC) to evaluate 3D asset production pipelines. The approved hybrid pipeline (TICKET-0011) uses AI generation primary + Blender decimation secondary.

Production assets created by the approved pipeline live at `game/assets/meshes/` and are NOT duplicated here.

---

## Contents

### `ai-gen/` — AI Generation Pipeline (Tripo3D) Output

Raw and Godot-imported assets from the Tripo3D API PoC (TICKET-0010).

| File | Description |
|------|-------------|
| `raw_mesh_*.glb` | Raw Tripo3D API output (100K-285K vertices, 9-16 MB each). Pre-decimation. |
| `mesh_*.glb` | Same raw meshes imported to Godot for visual review. |
| `mesh_*_Color_*.jpg` | Extracted PBR color maps (2048x2048, sRGB). |
| `mesh_*_NormalGL_*.jpg` | Extracted PBR normal maps (2048x2048, OpenGL format). |
| `mesh_*_ORM_*.jpg` | Extracted PBR ORM maps (2048x2048, Occlusion/Roughness/Metallic). |
| `poc_ai_review.tscn` | Godot review scene used during PoC evaluation. |
| `tripo_generate.py` | Python script for Tripo3D API asset generation. |

**Why not selected for production:** Raw meshes far exceed polygon budgets (100K-285K vs 2K-15K target). Required decimation pass in Blender, which is now part of the approved hybrid pipeline.

### `blender-python/` — Blender Python Pipeline Output

Assets generated by Blender Python scripting PoC (TICKET-0009).

| File | Description |
|------|-------------|
| `mesh_hand_drill.glb` | Procedural hand drill (flat-color PBR, ~3K tris). |
| `mesh_player_character.glb` | Procedural player character (flat-color PBR, ~6K tris). |
| `mesh_resource_node_scrap.glb` | Procedural resource node (flat-color PBR, ~2K tris). |
| `mesh_ship_exterior.glb` | Procedural ship exterior (flat-color PBR, ~4K tris). |
| `poc_asset_review.tscn` | Godot review scene used during PoC evaluation. |

**Why not selected as primary:** Scored 3.65 vs AI gen 3.55 in weighted evaluation, but margin (0.075) was within the 0.3 hybrid threshold. AI gen provides better visual quality with PBR textures; Blender provides reliable low-poly geometry. Hybrid approach approved.

### `misc/` — Other M2 Experiment Artifacts

One-off test assets created during early M2 experimentation.

| File | Description |
|------|-------------|
| `space_suit_character.glb` | Early character prototype from Blender scripting exploration. |
| `test_cube.glb` | Pipeline validation test geometry. |

## Retrieval

To reuse any artifact from this archive in a future sprint:

1. Copy the file into the appropriate `game/assets/` subdirectory
2. Godot will auto-import on next editor launch
3. If reusing AI gen meshes, apply the Blender decimation pass per `docs/art/3d-pipeline-sop.md`
