Core Philosophy

Scanner-First Design: Players discover before they act. Blind harvesting is inefficient by design.

Nomadic Survival: The ship moves constantly. Every destination is purposeful. Settling in place is never the goal.

Automation as Reward: Early game is manual and hands-on. Mid-game unlocks drones. Late game runs complex systems autonomously.

Mining Loop Design Principles

No Blind Clicking: Every mining decision is informed by scanner data. The minigame is optional but rewards player skill.

Purity Matters: High-purity deposits are rarer and found deeper. A 5-star vein is a meaningful discovery worth navigating to and defending.

Tool Gating: Each deposit tier requires a specific tool. Wrong tool = no extraction. Progression is tied to unlocking better tools.

Automation Preserves Skill: Mining drones execute baseline extraction (no minigame). Manual mining always outperforms drones on a per-deposit basis.

Early Game: "The Foundation"

  • Manual resource extraction
  • Hand Drill only
  • Basic scanner (Tier 1)
  • Small ship: minimal modules
  • Low fuel capacity
  • Restricted to low-threat biomes
  • Focus: Learn scanner-mining loop
  • Gate: First automation component

Mid Game: "The Expansion"

  • Mining drones unlock
  • Auto-smelter available
  • Multiple tool tiers unlocked
  • Scanner upgraded
  • Ship grows modularly
  • Medium-threat biomes accessible
  • Focus: Optimize, explore, stockpile
  • Gate: Mega-Project tech node

Late Game: "The Reckoning"

  • Mega-Project nodes only
  • Tier 4 tools available
  • High-threat zones required
  • Full ship optimization needed
  • Complex automation chains
  • Rare materials farming
  • Focus: Execute Mega-Project
  • Gate: Naer-Reth activation = WIN

Progression Gates & Win Conditions

First Cycle Win ("The Tragedy"): Complete Naer-Reth activation. This restores the planet's ecology but drains all energy (including the player's). The player does not escape. The ending is always tragic on the first playthrough.

Second Cycle (New Game+): Players who complete the first cycle carry meta-knowledge. A hidden path becomes available: modifying the Naer-Reth to prevent the destruction cycle. This requires 2× total resources and is significantly harder.

Loss Condition (Ship Destruction): Integrity reaches 0% → ship is destroyed. Tech tree and schematics persist. Resources and modules are lost. Player respawns at the last safe location with a starter ship configuration and rebuilds faster because they know everything.

No Attrition Loss: Starvation is NOT a loss condition. No forced survival timers. The game does not punish players for pausing exploration.

Tech Tree Structure

Early/Mid Game: Broad and flexible → Player chooses their path → Multiple viable strategies Late Game: Converges to Mega-Project spine → Single required progression path → All nodes enable automation, survivability, or Mega-Project progress → No dead-end nodes (every unlock has value) Mega-Project Nodes: → Distributed across world's biomes → Requires high-threat zone exploration → Completion = game win condition
